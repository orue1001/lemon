# 포트폴리오 웹사이트 기획서 (Project Lemon)

## 1. 프로젝트 개요
*   **프로젝트명**: Lemon Portfolio
*   **목적**: 개인의 기술 역량과 프로젝트 경험을 효과적으로 보여주는 웹 포트폴리오 구축
*   **타겟 사용자**: 채용 담당자, 클라이언트, 동료 개발자

## 2. 주요 목표
1.  **직관적인 정보 전달**: 사용자가 원하는 정보(기술 스택, 프로젝트 등)에 빠르게 접근할 수 있도록 함.
2.  **반응형 디자인**: 데스크탑, 태블릿, 모바일 등 모든 기기에서 최적화된 화면 제공.
3.  **성능 최적화**: 빠른 로딩 속도와 부드러운 인터랙션 제공.
4.  **개성 표현**: 'Lemon'이라는 프로젝트명에 걸맞은 상큼하고 현대적인 디자인 아이덴티티 구축.

## 3. 기술 스택 (제안)
*   **Core**: React 19, Vite
*   **Language**: JavaScript (추후 TypeScript 전환 고려 가능)
*   **Styling**: CSS Modules 또는 Tailwind CSS (선택 필요)
*   **Animation**: Framer Motion (부드러운 화면 전환 및 인터랙션)
*   **Deployment**: Vercel 또는 Netlify

## 4. 사이트맵 및 주요 기능

### 4.1. 홈 (Home)
*   **Hero Section**: 간단한 자기소개 및 인상적인 캐치프레이즈.
*   **Call to Action (CTA)**: '프로젝트 보기' 또는 '연락하기' 버튼.

### 4.2. 소개 (About Me)
*   **프로필**: 사진, 이름, 직군.
*   **소개글**: 개발 철학, 강점, 경험 요약.
*   **Skills**: 사용 가능한 기술 스택 아이콘 또는 리스트 시각화.
*   **이력 (Timeline)**: 학력 및 경력 사항 (선택 사항).

### 4.3. 프로젝트 (Projects)
*   **갤러리/그리드 뷰**: 프로젝트 썸네일 카드 리스트.
*   **필터링**: 기술 스택별, 카테고리별 프로젝트 필터링 기능.
*   **상세 모달 또는 페이지**:
    *   프로젝트 설명, 주요 기능, 기여도.
    *   사용 기술, 트러블 슈팅 경험.
    *   GitHub 레포지토리 링크 및 데모 사이트 링크.

### 4.4. 연락처 (Contact)
*   **연락 정보**: 이메일, LinkedIn, GitHub 링크.
*   **문의 폼**: 이메일 바로 보내기 기능 (EmailJS 등 활용).

## 5. **방명록 기능 구현**
* ID / Password 입력칸 (삭제용)
* supabase를 이용해서 데이터베이스와 연동

## 6. 디자인 컨셉 (제안)
*   **테마**: Clean, Minimal, Modern.
*   **컬러 팔레트**:
    *   Primary: #EBF222 (포인트 컬러)
    *   Tertiary: #F2E307 
    *   Background: #D0D9D6 / #8FA6A6(다크 모드 지원 고려)
    *   Text: #0D0D0D
*   **폰트**: 가독성이 좋은 산세리프 폰트 (예: Pretendard, Inter).

## 7. 필요한 이미지
* nanobanana를 이용한 생성형 이미지 생성


## 8. 개발 단계 (Roadmap)
1.  **기획 및 디자인**: `PLAN.md` 확정, 와이어프레임 구상.
2.  **환경 설정**: Tailwind CSS, 라우터 등 라이브러리 설치.
3.  **UI 컴포넌트 개발**: 헤더, 푸터, 레이아웃, 공통 버튼 등.
4. **페이지 구현**: Home -> About -> Projects -> Contact 순서로 개발.

6.  **콘텐츠 채우기**: 실제 프로젝트 데이터 및 이미지 추가.
7.  **배포 및 최적화**: 성능 테스트 및 호스팅.
